Firebase Task
--------------------------------------------

Esta es una app basica para crear tares del día a día.
Fue programada principamente con  React.js + Next.js + Express.js + y Firebase.js

## Instalation
    git clone https://github.com/Duwahiner/firebase-task
    npm install
    yart add
.
## Deveploment
    npm run dev

## Production
    npm run build
    npm start

    Disfrute de la app.
